# 🥗 NutriBot – AI-Powered Nutrition Agent

An intelligent AI-powered nutrition assistant built as part of the **IBM SkillsBuild AICTE Internship (Edunet Foundation)** using **IBM watsonx.ai Granite Models**. NutriBot provides personalized nutrition guidance, AI-powered meal planning, BMI analysis, calorie tracking, and interactive health coaching through a modern and responsive web interface.

Designed with a focus on Indian dietary habits, family health management, and accessibility, the application combines Artificial Intelligence with an intuitive user experience to help users make healthier food choices.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Soumit2k05/IBM-EDUNET-SOUMIT.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file and add:

```env
IBM_API_KEY=YOUR_API_KEY
WATSONX_PROJECT_ID=YOUR_PROJECT_ID
WATSONX_URL=https://us-south.ml.cloud.ibm.com
WATSONX_MODEL_ID=ibm/granite-3-8b-instruct
```

### Run

```bash
python app.py
```

Open:

```
http://localhost:5000
```
